Algorithm Analysis Deep Dive

Computer Science • March 15, 2025

Today we covered Big O notation and its practical applications. The concept of time complexity really clicked when we analyzed different sorting algorithms.

Key takeaways from today's lecture:

• Bubble sort: O(n²) - simple but inefficient for large datasets

• Merge sort: O(n log n) - much better performance

• Quick sort: O(n log n) average case, but O(n²) worst case

The professor's example using race cars really helped visualize how different algorithms scale with input size.
